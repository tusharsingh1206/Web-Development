Types of selector

  1. Element Selector: Same type of element is included.
  2. Universal Selector: Whole page is included.
  3. ID Selector: Unique element is included with unique id.
  4. Class Selector: elements with same class is included.
  5. Group Selector: If we want to give same property to multiple element then we use it.
