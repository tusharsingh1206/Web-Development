Transform Property

  1. Rotate is used to rotate the box by using the degree

  2. Scale is used to increase the size of the box by using the integer it will be 2 or 3,2 means twice.

  3. Translate is used to increase the size of the box in the x and in the y axis.

  4. Skew is used to flaten (Pichkana) the box.
