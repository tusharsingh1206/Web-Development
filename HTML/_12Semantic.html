THIS ALL TAG IS SEMANITIC TAG

  1. Header Tag

    2. The header tag is used to define the head of the page.

  2. Section Tag

    • The section tag is used to define the section in our page.
    • For example:
    • Technoligies:HTML,CSS,JAVASCRIPT
    • course OS/DBMS
    • Hobbies playing,coding

  3. Article Tag

    • This tag is used to show something in the page which is independent from our page but related to the page.
    • For example:
    • Lorem ipsum dolor sit amet consectetur adipisicing elit. A minus voluptatem sequi nobis ipsa totam est suscipit quasi sit, modi sint perferendis, blanditiis delectus dolorem necessitatibus laborum ullam aliquid explicabo.

  4. Aside Tag

    • This tag is used to show something which is not much important for our page .
    • For Exapmle:
    • Recommended books
      • S.Chand
      • RS.Aggrawal

  5. Footer Tag

    • This tag is generally used to give the link and usually shown the link at the bottom such as:
    • Linkdien
    • Hackerrank
    • Github
    • Quora
